- AM 9:00 ~ AM 10:30
- 스터디 종료 후 12시까지 1인당 질문 2개 남기기

- 질문 받은 당사자가 해당 질문에 대한 내용을 정리하여 PR
- 브랜치는 각자의 이름 적기
